INSTRUCCIONES PARA INSTALAR Y EJECUTAR
=======================================

Si tienes problemas para ejecutar el programa, sigue estos pasos:

PASO 1: INSTALAR PYTHON 3.11 (IMPORTANTE)
-----------------------------------------
El error que aparecio es porque instalaste una versio muy nueva de Python (probablemente la 3.13) que aun no es compatible con todas las librerias.

1. Desinstala el Python que tienes ahora.
2. Ve a este link para descargar **Python 3.11**:
   https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe
   (Ese enlace descarga la versión exacta que funciona).
3. Ejecuta el archivo descargado.
4. >>> ¡IMPORTANTE! MARCA LA CASILLA "Add Python.exe to PATH" antes de dar click en Install. <<<


PASO 2: EJECUTAR EL PROGRAMA
----------------------------
1. En esta carpeta, busca el archivo llamado "iniciar_programa.bat" (puede verse solo como "iniciar_programa").
2. Dale doble click.
3. Se abrirá una ventana negra que instalará todo lo necesario automáticamente.
4. Espera a que termine. La aplicación se abrirá en tu navegador.


Nota: La primera vez puede tardar unos minutos instalando las librerías. Las próximas veces será rápido.


PARA MAC (AMIGOS CON MACBOOK)
=============================
IMPORTANTE: Si te sale error de "Software malicioso" o "No se pudo verificar", lee el archivo "LEEME_MAC.txt".

1. INSTALAR PYTHON
   - Intenta abrir "descargar_python_mac.command".
   - Si no te deja: Click Derecho > Abrir > Botón "Abrir".
   - Sigue las instrucciones para instalar Python.

2. EJECUTAR EL PROGRAMA
   - Abre "iniciar_programa_mac.command".
   - (Si no te deja: Click Derecho > Abrir > Botón "Abrir").
   - IMPORTANTE: Dale permisos de ACCESIBILIDAD cuando te los pida.
     -> Ajustes > Privacidad > Accesibilidad > Activa "Terminal".
     -> Esto es obligatorio para que el programa pueda escribir en WhatsApp por ti.
